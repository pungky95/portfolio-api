# Portfolio

Portfolio is REST API using MySQL database built with Deno

## Installation

Denon installation


```bash
deno install --allow-read --allow-run --allow-write --allow-net -f --unstable https://deno.land/x/denon@v2.2.1/denon.ts
```

## Usage

```bash
denon start
```
